Custom-grid Sections

These sections are wonderful for presenting information in a non-linear fashion. This is great for using as a splash section on a homepage.

Separate CSS Sheet

Custom-grid sections have their own separate CSS file (custom-grids.css). This way, the custom sections of the CSS file is separate from the rest of the Unilay CSS.

Responsiveness Enabled

Custom sections are not responsive by default, but responsiveness can be added in through the custom CSS file.

Infinite Customization

This is one of the greatest features introduced in Unilay 4, helping projects distinguish themselves from one another.

Welcome to Unilay

Unilay is a universal web layout for projects at sensworks.ca and theonline.app. The aim of Unilay is to provide a uniform look across projects and to take away the hassle of producing a new design for every single app.

This is the 4th iteration of Unilay, with a new focus on flexibility, simplicity, and integration with content management systems and flask.

Confined Sections

Some sections are not ruled by a CSS grid, but simply confined to a maximum width. This one has a maximum width of 1000px. It is recommended to confine the width of elements so that the viewer does not have to constantly move their eyes from the left of the screen to the right side, especially on bigger screens.

Quick Tip: Text with a big red border on the left can be achieved with the class="left-border-decoration". On a separate note, the confined sections are achieved with the class="confined-1000".

Auto-columned class

It might also be advisable to split up text-heavy blocks into columns. Any div with the columns-n class will automatically divide up its content into n columns. This makes text easier to read especially on bigger screens. When used in mobile view, column-n class elements will display everything in a single column. Try not to mix up text with divs and images in an element with the columns-n class, as that makes equal divisions difficult. Any divs inside a column-n class element will not break, but retains its grouping as a block.

The auto-columned class is class="columns-3"

Images

All images, regardless of size, will not exceed its parent container. class="full-width" can also be added to images so even smaller images will take up the entire space. This class gives images the "width: 100%;" property. Images are also given 2rems of margin below them.

Sections with spaces reserved for sidebars

Some sections benefit from having a sidebar attached to it. A sidebar can contain navigation links, relevant instructions or help messages, contain notifications, etc.

This is achived by adding class="with-sidebar" to the containing block-level element, then adding class="section-main" to the main DIV. An aside element should be used to contain the sidebar.
